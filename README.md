# Railway Ticket Booking

React/Vite навчальний проєкт для лабораторних робіт 9-10: модель системи
продажу та бронювання залізничних квитків.

## Запуск

```bash
npm install
npm run dev
```

## Збірка

```bash
npm run build
```

## Можливості

- список рейсів у вигляді карток;
- пошук за номером поїзда або маршрутом;
- фільтрація за типом поїзда та сортування за часом, ціною або тривалістю;
- вибір вагона і декількох місць;
- швидкий добір двох сусідніх місць;
- форма бронювання з валідацією;
- збереження бронювань у `localStorage`.
